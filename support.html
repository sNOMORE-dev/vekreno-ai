FAQ

Common questions and answers.

How do I log in to my account?

Go to the signin page and enter the same email and password you used during signup. If the details match what is stored in localStorage, the app will create an active session and send you to the dashboard.

Why am I being redirected to signin?

Protected pages check whether a current user exists before showing content. If no session is found, the app sends you back to the signin page so the page stays private.

How do I generate an exam paper?

Open the Generate page, fill in the title, subject, topic, difficulty, and number of questions, then click Generate. The app.js file can create a text preview and show it in the output panel.

Where are saved exam papers stored?

Saved papers are stored in the browser using localStorage and linked to the current user account. They remain available after refresh because localStorage persists across browser sessions.

Can I delete or edit a saved paper?

You can add those features later by updating the saved exams array in app.js. A delete button can remove an item from the array, save the updated data, and re-render the list.

How do points and levels work?

Your app can increase points when a user generates or saves an exam. The level can be calculated from points, such as one level for every 100 points, or any system you choose.

How do I upgrade my plan?

Open the Upgrade page and choose a plan card. For now, the page can mark the account as upgraded in localStorage, and later you can connect it to a real payment provider.
